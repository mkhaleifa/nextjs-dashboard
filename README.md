# 📊 Next.js Dashboard App

A full-stack dashboard application built with **Next.js App Router**.
This project demonstrates modern web development practices including authentication, database integration, protected routes, and dynamic data rendering.

## 🧠 Features

* 🔐 Authentication (Login / Protected Routes)
* 📊 Dashboard analytics overview
* 🧾 Invoice management system
* 👥 Customer data handling
* ⚡ Static & Dynamic Rendering
* 🧩 Server & Client Components
* 🗄️ Database integration (PostgreSQL)
* 🌐 Deployed on Vercel

---

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **Frontend:** React, Tailwind CSS
* **Authentication:** Auth.js / NextAuth
* **Database:** PostgreSQL
* **ORM / Queries:** (Drizzle / Prisma / SQL)
* **Deployment:** Vercel

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/nextjs-dashboard.git
cd nextjs-dashboard
```

Install dependencies:

```bash
pnpm install
```

---

## 🗄️ Environment Variables

Create a `.env` file:

```
POSTGRES_URL=your_database_url
AUTH_SECRET=your_secret_key
```

---

## 🌱 Seed Database

Run the seed script:

```bash
pnpm dlx tsx scripts/seed.ts
```

This will create demo users, invoices, and customers.

---

## ▶️ Run Development Server

```bash
pnpm dev
```

App runs on:

```
http://localhost:3000
```
